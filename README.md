
# Velvet

The end goal is for this to be an opinionated meta-framework for building fullstack
hypermedia driven web applications in Rust using the SASHA stack (SQLite, Axum, Shoelace, HTMX, AlpineJS)

### Dependencies

- Make
- PNPM
- Rust 2024
- Nodejs 23
- SeaORM CLI
